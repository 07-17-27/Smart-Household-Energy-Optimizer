# Smart Energy Consumption Forecasting System ⚡
This project predicts next month's electricity bill using Machine Learning and provides smart recommendations to reduce energy consumption.

## Features
- Electricity bill prediction using XGBoost
- Appliance-wise cost analysis
- Recommendation system (reduce usage in minutes/day)
- Streamlit web application
  
## Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit

## Model Performance
- R² Score: 0.9195

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
